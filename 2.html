HTML 메모

HTML5 관련 설정

  1. HTML5는 반드시! 맨 위에 !doctype html를 적어서 문서 형식을 HTML5로 지정
  2. 모든 줄은 html 태그 안에 위치
  3. head 태그는 외부 파일의 참조, 메타데이터 등이 위치, 브라우저에 표시 X
  4. body 태그에는 브라우저에 출력할 부분을 입력

HTML 기본 문법

  1. HTML의 요소는 앞에 있는 태그를 시작태그, 뒤에 있는 종료태그로 나뉘고, 태그 사이에 위치한 콘탠츠로 구분됨
  2. 빈 요소를 가지는 태그도 존재

인터넷을 이용한 웹

  1. 영어로 고객은 클라이언트, 사업자는 서버
  2. 요청하는 컴퓨터는 클라이언트 컴퓨터, 응답하는 컴퓨터는 서버 컴퓨터라고 부름
  3. 클라이언트 컴퓨터에서 동작하는 웹을 웹 클라이언트, 서버 컴퓨터에서 동작하는 웹을 웹 서버라고 부름
  4. 인터넷에 연결된 컴퓨터 하나 하나를 호스트라고 함
  5. 이런 컴퓨터를 빌려주는 사업을 호스팅이라 함
  6. 웹 서버를 전문적으로 빌려주는 비즈니스를 웹 호스팅 업체라고 부름
  7. 이런 서비스 중 Github의 pages를 이용해서 인터넷에 웹을 서비스할 수 있음
  8. 웹 서버를 운영하는 것보다 웹 호스팅을 이용하면 쉬움, 하지만 인터넷의 원리가 감춰져 있음
  9. 웹 서버를 운영하기 위해서는 많은 노하우가 필요함
  10. 정적 웹 페이지 : 저장된 그대로 사용자에게 전달되는 웹 페이지
  11. 동적 웹 페이지 : 웹 서버가 데이터를 생성하거나 가공하여 전달해주는 웹 페이지

Github 웹 호스팅 이용 방법

  1. github에 가입하지 않았다면 sign up ( 회원가입 )을 하고 sign in ( 로그인 )한다.
  2. repository ( 저장소 )를 생성한다.
  3. Repository name ( 저장소 이름 )에 사용자.github.io를 입력해야 웹 호스팅을 이용할 수 있다.
  4. public으로 설정한다.
  5. Create repository ( 저장소 생성하기 ) 버튼을 누른다.
  6. uploading an existing file을 클릭
  7. 작업한 파일을 상단 박스에 추가한다.
  8. Add file via upload에는 파일을 변경할 때마다 적는 변경 사항을 적어주는 곳이다.
  9. Commit changes버튼을 누르면 업로드가 시작
  10. 상단의 settings버튼을 클릭
  11. 사이드 바에서 Pages 클릭
  12. Branch를 main으로 설정하고 save 클릭
  13. 그럼 자신이 올린 파일의 웹이 웹 호스팅 완료!
