HTML 메모

HTML5 관련 설정

  1. HTML5는 반드시! 맨 위에 !doctype html를 적어서 문서 형식을 HTML5로 지정
  2. 모든 줄은 html 태그 안에 위치
  3. head 태그는 외부 파일의 참조, 메타데이터 등이 위치, 브라우저에 표시 X
  4. body 태그에는 브라우저에 출력할 부분을 입력

HTML 기본 문법

  1. HTML의 요소는 앞에 있는 태그를 시작태그, 뒤에 있는 종료태그로 나뉘고, 태그 사이에 위치한 콘탠츠로 구분됨
  2. 빈 요소를 가지는 태그도 존재

인터넷을 이용한 웹

  1. 영어로 고객은 클라이언트, 사업자는 서버
  2. 요청하는 컴퓨터는 클라이언트 컴퓨터, 응답하는 컴퓨터는 서버 컴퓨터라고 부름
  3. 클라이언트 컴퓨터에서 동작하는 웹을 웹 클라이언트, 서버 컴퓨터에서 동작하는 웹을 웹 서버라고 부름
