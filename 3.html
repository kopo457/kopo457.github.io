CSS 메모

CSS를 이용한 색깔 바꾸기

  1. head 태그 안에 style 태그를 적기
  2. 변경할 태그 이름만 쓰고 뒤에 중괄호{} 안에 color:변경할 색상;을 적으면 변경하고자 하는 태그의 폰트 색상이 변경됨

CSS의 기본 문법

  1. 속성을 이용해서 원하는 디자인을 만들 수 있음
  2. HTML의 태그 안에 style이라는 속성을 이용해서 그 태그의 디자인만 변경할 수 있음
  3. color:red는 웹 브라우저에선 이 문법이 HTML인지 CSS인지 구분할 수 없음
  4. 그래서 CSS의 효과를 주고 싶으면 style이라는 HTML의 속성 안에 위치한 값이 CSS인지 구분할 수 있게 만듦.
  5. 선택자 : CSS의 효과를 어떤 태그에 줄 것인지 선택하는 문법
  6. 이 페이지에선 h1{} 이 부분이 선택자
  7. 선택자 안에 있는 문법을 효과라고 부름
  8. 태그 안에 속성으로 이용될 때는 선택자를 사용할 필요가 없음
  9. CSS 태그를 이용할 때는 효과 뒤에는 무.조.건! 세미콜론 ;을 붙여야 함!
  10. style 속성 안에 두 개 이상의 효과를 써야할 때도 세미콜론을 써야함!

CSS를 이용하는 2가지 방법

  1. style 태그를 이용해서 변경할 태그 모두 바꾼다.
  2. style 속성을 이용해서 하나의 태그만 바꾼다.
